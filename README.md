#Hard Mode

Version 1.0 of GHPC Hard Mode
Adds additional enemies to base game maps. Currently only supports Patton Pass and Abrams Alley.

Requires Melonloader 0.6.1 and [Vehicle Preloader](https://github.com/thebeninator/VehiclePreloader/releases/tag/1.0.0).

To install, go to releases and download the latest copy.

Known Bugs:

- Enemies not required to be killed to finish mission.
